# CadIME Modern (2025–2027)

基于 AutoCAD 2025 / 2026 / 2027（.NET 8.0 & .NET 10.0）托管运行时深度定制的纯被动防护型智能中英文输入法自动切换插件。

核心技术特色
  零残留，纯被动防御：彻底砍掉传统的 250ms 高频定时轮询心跳（Application.Idle），平时处于 绝对零残留静默状态。完全由 AutoCAD 官方四大生命周期事件（开始、结束、取消、失败）被动触发。
  硬件级焦点自愈网：直接向操作系统注册 SetWinEventHook 焦点挂钩，1 微秒内被动捕捉按 ESC 或文字编辑完瞬间的光标质变。
  CMDNAMES 忙时避让：焦点信号异步封送回 CAD 主 UI 线程（STA），独占读取内核变量 CMDNAMES。不为空时插件绝对闭嘴，从源头死锁普通命令（L/MOVE）拉线中途误切中文、提前切英文的内耗。
  多中文输入法全承认：全面摒弃古老越权的 IMM32 强刷状态键做法。采用位运算检查实际键盘布局低 16 位，只要是中文（0x0804 族群，如搜狗、微软拼音、五笔等），插件 100% 保持最高敬意、顺应用户手动修改，绝不反噬强刷。
  前置安全自检熔断：启动时单次检测系统是否配齐 0x04090409（美国英语）布局。缺失则首次弹窗引导，后续流式扫描历史日志，直接静默自封禁用退出，绝不重复弹窗骚扰用户。

极致内聚工程架构
  免注册 JSON 绿色跟随：白名单清单 CadIME.json 同目录就近跟随，解压即用。
  源码级大融合裸编译：彻底废除 .csproj 项目文件。由 Build2027.ps1 一键驱动物理雷达消杀净化，粉碎清除所有中间件 bin/obj 废旧缓存垃圾，直接使出微软底层 Roslyn 编译器真身，合体打包输出最纯净、无任何警告错误的通用免注册单体 DLL。

完美人机协同体验
1.  空白等待/普通绘图期：输入法稳如磐石留存英文，100% 守护绘图坐标与数值输入流畅度。
2.  特定命令打字期：中文打字行云流水，期间允许用户手任意手动乱切各种中文输入法厂商。
3.  完成回车/ESC取消瞬间：硬件钩子卡死分流，物理输入法 毫无挂滞、极速单次强制跳变退回英文状态，彻底降服输入法死锁顽疾。
